# 访问日志查看指南

## 📋 概述

访问日志记录了所有的刷卡记录和开门记录，包括成功和失败的访问尝试。通过日志可以：
- 查看谁在什么时间访问了哪个门禁
- 分析访问模式和异常行为
- 审计安全事件
- 生成访问报表

## 🎯 功能特性

### ✅ 已实现功能

- **实时统计** - 今日访问总数、成功率等
- **多条件筛选** - 按卡片、设备、时间、结果筛选
- **快速时间范围** - 今天、昨天、本周、本月
- **分页浏览** - 每页50条记录
- **导出功能** - 导出为CSV文件
- **详细信息** - 显示卡片、设备、时间、结果、来源

## 🚀 快速开始

### 1. 访问日志页面

1. 登录Web管理界面
2. 点击左侧导航栏的"访问日志"
3. 进入日志查看页面

### 2. 查看今日统计

页面顶部显示今日统计信息：
- **今日总访问**: 今天所有的访问次数
- **今日允许**: 成功通过的次数
- **今日拒绝**: 被拒绝的次数
- **今日成功率**: 成功率百分比

### 3. 使用快速筛选

点击快速筛选按钮：
- **今天**: 查看今天的记录
- **昨天**: 查看昨天的记录
- **本周**: 查看本周的记录
- **本月**: 查看本月的记录
- **全部**: 查看所有记录

### 4. 高级筛选

使用筛选条件：
- **卡片UID**: 输入卡片UID查找特定卡片的记录
- **设备ID**: 输入设备ID查找特定设备的记录
- **访问结果**: 选择"允许"或"拒绝"
- **开始时间**: 设置查询的开始时间
- **结束时间**: 设置查询的结束时间

### 5. 导出日志

1. 设置筛选条件（可选）
2. 点击"导出日志"按钮
3. 系统会下载CSV文件
4. 可以用Excel打开查看

## 📊 日志字段说明

### 时间
- **格式**: YYYY-MM-DD HH:MM:SS
- **说明**: 访问发生的时间
- **附加**: 显示相对时间（如"5分钟前"）

### 卡片信息
- **持卡人**: 卡片关联的姓名
- **UID**: 卡片的唯一标识符
- **显示**: 
  - 如果卡片已注册，显示姓名
  - 如果卡片未注册，显示"未知"

### 设备信息
- **设备名称**: 设备的友好名称
- **设备ID**: 设备的唯一标识符
- **显示**:
  - 如果设备已注册，显示名称
  - 如果设备未注册，显示"未知"

### 访问结果
- **允许** (绿色徽章): 访问成功，门已打开
- **拒绝** (红色徽章): 访问被拒绝，门未打开

### 来源
- **云端** (蓝色徽章): 通过云端验证
- **缓存** (灰色徽章): 通过设备本地缓存验证

### 原因
- 访问被拒绝的原因
- 常见原因：
  - "Card not found" - 卡片未注册
  - "Card disabled" - 卡片已禁用
  - "Outside time range" - 不在允许的时间范围内
  - "Outside time slot" - 不在允许的时间段内
  - "Device not allowed" - 设备不在允许列表中

## 💡 使用场景

### 场景1: 查看某人的访问记录

```
1. 在"卡片UID"输入框输入该人的卡片UID
2. 点击"搜索"按钮
3. 查看该人的所有访问记录
```

**用途**:
- 考勤统计
- 行为分析
- 异常调查

### 场景2: 查看某个门禁的使用情况

```
1. 在"设备ID"输入框输入设备ID
2. 选择时间范围（如"本月"）
3. 点击"搜索"按钮
4. 查看该门禁的所有访问记录
```

**用途**:
- 设备使用率分析
- 高峰时段分析
- 设备维护计划

### 场景3: 查看被拒绝的访问

```
1. 在"访问结果"下拉框选择"拒绝"
2. 选择时间范围（如"今天"）
3. 点击"搜索"按钮
4. 查看所有被拒绝的访问尝试
```

**用途**:
- 安全审计
- 异常行为检测
- 权限问题排查

### 场景4: 生成月度访问报表

```
1. 点击"本月"快速筛选按钮
2. 点击"导出日志"按钮
3. 打开下载的CSV文件
4. 使用Excel进行数据分析
```

**用途**:
- 月度报表
- 数据分析
- 管理决策

### 场景5: 调查安全事件

```
1. 设置事件发生的时间范围
2. 可选：输入相关的卡片UID或设备ID
3. 点击"搜索"按钮
4. 查看该时间段的所有访问记录
```

**用途**:
- 安全事件调查
- 时间线重建
- 证据收集

## 🔍 高级查询技巧

### 技巧1: 组合筛选

可以同时使用多个筛选条件：
```
卡片UID: 04A1B2C3D4E5F6
设备ID: door-001
访问结果: 拒绝
时间范围: 2024-05-01 到 2024-05-03
```

这样可以精确定位特定的访问记录。

### 技巧2: 时间范围查询

使用开始时间和结束时间进行精确查询：
```
开始时间: 2024-05-03 09:00
结束时间: 2024-05-03 18:00
```

查看工作时间内的所有访问。

### 技巧3: 部分UID匹配

如果不记得完整的UID，可以输入部分UID：
```
卡片UID: 04A1
```

系统会搜索包含"04A1"的所有UID。

### 技巧4: 重置筛选

如果筛选条件太多，点击"重置"按钮清空所有筛选条件。

## 📥 导出功能

### CSV文件格式

导出的CSV文件包含以下列：
```
时间,卡片UID,持卡人,设备ID,设备名称,访问结果,来源,原因
```

### 导出步骤

1. **设置筛选条件**（可选）
   - 只导出符合条件的记录
   - 不设置则导出所有记录

2. **点击导出按钮**
   - 点击页面右上角的"导出日志"按钮

3. **等待下载**
   - 系统会生成CSV文件
   - 浏览器会自动下载

4. **打开文件**
   - 使用Excel或其他表格软件打开
   - 文件名格式: `access_logs_YYYY-MM-DD.csv`

### 导出限制

- 单次最多导出10000条记录
- 如果记录超过10000条，建议分批导出
- 使用时间范围筛选来减少记录数量

### Excel处理

打开CSV文件后，可以：
- 使用Excel的筛选功能
- 创建数据透视表
- 生成图表
- 进行统计分析

## ⚠️ 注意事项

### 数据保留

- 日志数据会永久保存
- 建议定期导出备份
- 可以手动清理旧数据（需要数据库操作）

### 性能考虑

- 查询大量数据可能较慢
- 建议使用时间范围筛选
- 避免查询全部记录

### 隐私保护

- 日志包含敏感信息
- 只有管理员可以查看
- 导出的文件应妥善保管

### 时区问题

- 所有时间使用本地时区
- 导出的CSV文件也使用本地时区
- 跨时区使用需要注意时间转换

## 🔧 故障排查

### 问题1: 没有显示日志

**可能原因**:
- 筛选条件太严格
- 时间范围不正确
- 确实没有访问记录

**解决方法**:
1. 点击"重置"按钮清空筛选
2. 选择"全部"时间范围
3. 检查是否有设备在线

### 问题2: 导出失败

**可能原因**:
- 记录数量太多
- 网络连接问题
- 浏览器限制

**解决方法**:
1. 缩小时间范围
2. 添加更多筛选条件
3. 检查网络连接
4. 尝试其他浏览器

### 问题3: 时间显示不正确

**可能原因**:
- 浏览器时区设置
- 服务器时区设置

**解决方法**:
1. 检查浏览器时区设置
2. 检查系统时间
3. 联系管理员检查服务器时区

### 问题4: 统计数据不准确

**可能原因**:
- 数据还在同步中
- 缓存问题

**解决方法**:
1. 点击"刷新"按钮
2. 清除浏览器缓存
3. 重新登录
