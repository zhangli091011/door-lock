# 设备注册快速参考

## 🚀 快速开始

### 1️⃣ 注册设备（Web界面）

```
登录 → 设备管理 → 注册设备 → 填写信息 → 保存凭证
```

**必填字段**:
- 设备ID: `door-001` (字母、数字、下划线、连字符)
- 设备名称: `前门门禁` (1-100字符)

**可选字段**:
- 位置: `一楼大厅`
- MAC地址: `AA:BB:CC:DD:EE:FF`

### 2️⃣ 配置ESP32

编辑 `esp32-firmware/config.h`:

```cpp
#define DEVICE_ID "door-001"
#define API_KEY "从Web界面复制"
#define SECRET_KEY "从Web界面复制"
#define SERVER_URL "https://your-domain.com"
```

### 3️⃣ 上传固件

```
Arduino IDE → 选择ESP32-S3 → 上传
```

## 📋 API快速参考

### 注册设备

```bash
POST /api/devices
Authorization: Bearer <JWT_TOKEN>

{
  "device_id": "door-001",
  "name": "前门门禁",
  "location": "一楼大厅",
  "mac_address": "AA:BB:CC:DD:EE:FF"
}
```

**响应**: 返回 `api_key` 和 `secret_key`

### 更新设备

```bash
PUT /api/devices/door-001
Authorization: Bearer <JWT_TOKEN>

{
  "name": "前门门禁（已更新）",
  "enabled": true
}
```

### 获取设备列表

```bash
GET /api/devices
Authorization: Bearer <JWT_TOKEN>
```

## ⚠️ 重要提示

### 🔐 安全注意事项

1. **Secret Key只显示一次** - 注册后立即保存
2. **设备ID不可修改** - 注册前仔细确认
3. **妥善保管凭证** - 不要提交到版本控制
4. **定期检查设备** - 监控在线状态

### ✅ 验证规则

| 字段 | 规则 |
|------|------|
| 设备ID | 字母、数字、下划线、连字符，1-50字符 |
| 设备名称 | 1-100字符 |
| 位置 | 0-200字符 |
| MAC地址 | XX:XX:XX:XX:XX:XX 格式 |

### 🔄 在线状态

- **在线**: 最后在线时间 < 5分钟
- **离线**: 最后在线时间 ≥ 5分钟

## 🛠️ 故障排查

### 问题：设备ID已存在
**解决**: 使用不同的设备ID

### 问题：MAC地址已存在
**解决**: 检查MAC地址或使用不同的MAC

### 问题：设备显示离线
**解决**: 
1. 检查ESP32网络连接
2. 确认API Key和Secret Key正确
3. 查看ESP32串口输出

### 问题：无法查看Secret Key
**解决**: 
- 如已保存，使用已保存的密钥
- 如未保存，需重新注册设备
